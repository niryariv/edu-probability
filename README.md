# עולם ההסתברות - Probability Education Website

An interactive probability learning website for ages 10-12, built with pure HTML, CSS, and JavaScript.

## Live Site

[https://niryariv.github.io/edu-probability/](https://niryariv.github.io/edu-probability/)

## Modules

1. **Coins (מטבעות)** - Explore heads and tails, fair vs. unfair coins, and basic probability concepts through coin flip simulations.
2. **Dice (קוביות)** - Roll dice and discover outcomes, sample spaces, and the law of large numbers with interactive dice experiments.
3. **Spinners (גלגלים)** - Spin customizable probability wheels and learn how area relates to likelihood of outcomes.
4. **Cards (קלפים)** - Draw from a deck of cards to understand equally likely events and compound probability.
5. **Bag (שקית)** - Draw colored balls from a bag to explore experimental vs. theoretical probability.
6. **Quiz (חידון)** - Test your probability knowledge with an interactive quiz covering all module concepts.

## Tech Stack

- Pure static HTML, CSS, and JavaScript - no build step or dependencies required
- Deployed via GitHub Actions to GitHub Pages
- Responsive design suitable for tablets and desktops
